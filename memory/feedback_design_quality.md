For ALL Munshi design work, the ONLY visual reference is the brand bible at brand-showcases-nine.vercel.app. Never improvise or use generic SaaS styling. Japanese minimalism, billion-dollar quality.

**Why:** User explicitly rejected a deck that used approximate/generic styles instead of the exact brand bible tokens. Called it "absolute garbage" and "slop." Also flagged concepts D/E/F as tacky — too many effects, missing the logo, not clean enough.

**How to apply:**
- Always fetch and study the brand bible site BEFORE any design work
- Section titles: 36px DM Sans 700, letter-spacing -0.025em, line-height 1.15
- Body/prose: 15px Inter, line-height 1.75, color #2D3039, max-width 640px
- Cards: white bg, 1px solid #DFE2E8, border-radius 12px, shadow-card (0 1px 3px rgba(27,107,74,.08)), hover lift translateY(-1px) + shadow-raised
- Tables: th 10.5px uppercase weight 600, #9CA1AD color, bg #F7F8FA; td 13px, padding 13px 16px
- Badges: 11.5px weight 600, pill 9999px, padding 3px 10px
- Buttons: 13.5px weight 600, padding 9px 18px, radius 8px
- Page padding: 80px
- Shadows ALWAYS Neelam blue-tinted (rgba(27,107,74,...))
- 3px gradient accent line at top (linear-gradient(135deg, #1B6B4A, #2A9D8F))
- Spacing: 4px grid (4,8,12,16,24,32,48,64,80)
- Border radius: sm 4px, md 8px, lg 12px, xl 16px, full 9999px
- Hover: always lift+shadow, 0.15s ease default
- Financial data: IBM Plex Mono, tabular-nums, Indian format
- Kesar Gold: sparingly (AI, premium, accents ONLY)
- Reference Stripe, Linear, Notion, Vercel, Apple design. Ma (negative space), Kanso (simplicity), Shibui (understated elegance).
