Munshi brand bible v3.0 approved on April 5, 2026.

**Direction:** A — Stripe Neelam (surgical precision, global fintech)
**Key rules:**
- NO Devanagari script — all Hindi in Roman/English alphabet only
- Logo: Open-book monogram (Neelam Green gradient #1B6B4A → #2A9D8F), "Munshi." wordmark (DM Sans 700, gold dot #E8A317)

- Typography: DM Sans 400 display, Inter body, IBM Plex Mono financial

- 8px radius (never pills), no glow/gradients/stock photos
